# 🔒 SecureXPro

> A desktop-based vulnerability assessment tool that enables security teams to scan, identify, exploit, and report vulnerabilities across web applications and networks.

---

## 📌 About The Project

SecureXPro is a professional-grade cybersecurity tool designed for admins and security agents. It automates the process of scanning targets, detecting known vulnerabilities, mapping exploits, assigning risk scores, and generating detailed security reports — all from a clean desktop interface.

---

## ✨ Features

- 🌐 **Web & Network Scanning** — Run targeted scans against domains and network hosts
- 🐛 **Vulnerability Detection** — Identify known CVEs and security weaknesses
- 💥 **Exploit Mapping** — Match detected vulnerabilities to known exploits
- 📊 **Risk Scoring** — Automatically assign severity scores (Critical / High / Medium / Low)
- 📄 **Report Generation** — Export structured security reports for review
- 👥 **Role-Based Access** — Separate admin and agent access levels
- 🔐 **Secure Authentication** — JWT-based login system

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- Python 3.x
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/emanbangash08/securexpro.git

# Navigate to project
cd securexpro

# Install dependencies
npm install

# Run in browser (development)
npm start

# Build desktop application
npm run electron-build
```
